C++ Standards Committee
Library Working Group

This is the home page for the Library Working Group (LWG) of the C++ Standards Committee, JTC1/SC22/WG21. It provides easy access to current drafts of documents that may have changed since the last official document release. Drafts have "D" document numbers, while official versions have "N" numbers and can be found on the official WG21 web site.
